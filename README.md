## インストール 

[スプレッドシート拡張 - Chrome ウェブストア](https://chrome.google.com/webstore/detail/sheets-tabs-on-top/lbhlhhckfpdpafckdiklcbamkmogjdjc)

# Dev
## Install
```
$ yarn install
```

## Development
```
$ yarn dev chrome
$ yarn dev firefox
$ yarn dev opera
$ yarn dev edge
```

## Build
```
$ yarn build chrome
$ yarn build firefox
$ yarn build opera
$ yarn build edge
```
